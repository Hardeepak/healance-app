# 🌿 Helance - Hackathon Project

Welcome to **Helance**: An AI-Powered Anonymous Hub for Youth Wellbeing & Career Progression.

## 👥 The Team & Roles
* **Role 1 (Frontend):** Flutter & UI Design.
* **Role 2 (Backend):** Database schema, API routes, user data.
* **Role 3 (AI Systems):** LLM integration (Safety Intercept, Triage Report, Career Pivot logic).
* **Role 4 (Integrator):** GitHub Repo Management, `flutter_map` Integration, & Deployment.
