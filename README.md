# 🌿 Héalance - Hackathon Project

Welcome to Héalance : An AI-Powered Anonymous Hub for Youth Wellbeing & Career Progression. 

## 📂 Project Structure
* `/frontend`: UI, React/Next.js components, Tailwind CSS. (Owned by Role 1)
* `/backend`: Database connections, API routes, and AI scripts. (Owned by Roles 2 & 3)

## 🚦 Git Workflow Rules (READ THIS)
1. **Never push directly to `main`.** The `main` branch is our live, deployable code.
2. When you start working, branch off `dev`. 
   *(e.g., `git checkout -b frontend-feed-ui`)*
3. When your feature is done, push your branch and tell Role 4 to review and merge it into `dev`.
4. **DO NOT UPLOAD `.env` FILES.** Keep your API keys local.
